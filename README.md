# shaif.dev

Personal portfolio site for Shaif Mohammad Reasat. Built with [Astro](https://astro.build) + [Tailwind CSS](https://tailwindcss.com), shipped as a static site.

## Quick start

```bash
npm install
npm run dev          # http://localhost:4321
```

## Build & preview

```bash
npm run build        # outputs to dist/
npm run preview      # serve dist/ locally
```

## Stack

- **Astro 5** (static output)
- **Tailwind CSS 3** (class-based dark mode, custom font sizing)
- **TypeScript** (strict)
- **Inter** + **JetBrains Mono** via Google Fonts
